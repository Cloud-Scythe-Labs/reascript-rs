This project is a work in progress as-needed for the ReaCloud REAPER extension plugin, and is not feature complete.
Features are likely to be added periodically as the project develops, though pull-requests are welcome!

To use the crate in your REAPER extension, add it to your `Cargo.toml`:

```toml
[dependencies]
reascript = { git = "https://github.com/Cloud-Scythe-Labs/reascript-rs.git" }
```
